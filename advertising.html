Sub System

광고

진입 경로

홈
→ 서브 시스템
→ 광고

광고 시스템에서는 광고 이미지를 만들고, 광고 일정을 등록하며, 현재 편성된 광고를 확인한다.

기본 구조

광고 시스템은 다음 세 단계로 구성된다.

광고 이미지 생성
→ 광고 일정 생성
→ 광고 스케줄 확인

광고는 지정한 시간 동안 노출되며, 설정한 효과에 따라 손님 반응과 상품 판매에 영향을 준다.

광고 이미지 생성

광고 이미지 화면에서는 보유한 광고 소재를 확인하고 사용할 이미지를 선택한다.

주요 기능

  • 광고 이미지 목록 확인
  • 이미지 미리보기
  • 광고 소재 선택
  • 페이지 이동
  • 새 광고 이미지 생성
  • 선택 취소

광고 이미지는 상품, 세트 상품, 회사, 이벤트와 연결하여 사용할 수 있다.

광고 이미지 목록
→ 이미지 선택
→ 미리보기 확인
→ 생성

광고 일정 생성

광고 일정 생성 화면에서는 광고 소재와 노출 시간을 설정한다.

광고 미리보기

광고에 사용할 이미지를 선택한다.

사용자 이미지
기본 광고 소재

일정 설정

  • 날짜
  • 시작 시간
  • 종료 시간
  • 광고 효과

광고 정보

  • 광고 제목
  • 광고 소재 이름

설정을 완료하면 광고 일정을 생성할 수 있다.

광고 이미지 선택
→ 광고 제목 입력
→ 날짜 설정
→ 시작·종료 시간 설정
→ 광고 효과 선택
→ 일정 생성

광고 스케줄

광고 스케줄 화면에서는 등록된 광고 일정과 현재 상태를 확인한다.

광고 미리보기

선택한 광고의 전체 이미지를 표시한다.

광고 정보

  • 실행 날짜
  • 실행 시간
  • 캠페인 효과

일정 목록

  • 시간
  • 광고 소재 이름
  • 해상도
  • 재생 길이
  • 상태

상태

Assigned
Unassigned

`Assigned`는 해당 시간에 광고가 배정된 상태다.

`Unassigned`는 아직 광고가 배정되지 않은 상태다.

날짜 선택
→ 시간대별 광고 일정 확인
→ 광고 선택
→ 미리보기와 효과 확인

광고 적용 흐름

광고 이미지 생성
→ 광고 일정 등록
→ 지정 시간에 광고 노출
→ 손님 반응 변화
→ 상품 판매 결과 변화

광고 효과는 다음 요소에 영향을 줄 수 있다.

  • 방문자 수
  • 상품 선호도
  • 구매 빈도
  • 특정 상품 판매량
  • 회사 인지도
  • 캠페인 효과

메뉴 설명

광고

광고 이미지를 만들고 날짜와 시간을 지정해 광고 일정을 등록합니다. 편성된 광고의 소재, 해상도, 재생 시간, 상태와 캠페인 효과를 확인할 수 있습니다.
