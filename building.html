진입 경로

→ 빌딩

빌딩 메뉴에서는 농장, 공장, 상점, 개미굴, 연구소에 설치된 시설을 한 번에 확인한다.

기본 구성

빌딩 화면은 설치 구역별 탭으로 나뉜다.

농장
공장
상점
개미굴
연구소

구역을 선택하면 해당 구역에 설치된 시설 목록과 현재 적용 중인 효과가 표시된다.

설치 구역 선택

화면 위쪽에서 확인할 구역을 선택한다.

구역 종류

  • 농장
  • 공장
  • 상점
  • 개미굴
  • 연구소

    • 선택한 구역에 따라 왼쪽 대표 이미지와 오른쪽 시설 목록이 변경된다.

구역 요약

화면 왼쪽에는 현재 선택한 구역의 대표 이미지와 전체 시설 현황이 표시된다.

표시 정보

  • 설치된 건물 수
  • 풍년 효과 적용 수
  • 이벤트 효과 적용 수
    • TOTAL FACILITIES
→ 설치된 건물 수
→ 풍년 효과 적용 중
→ 이벤트 효과 적용 중

    설치된 시설 목록

    화면 오른쪽에는 현재 구역에 설치된 시설이 목록으로 표시된다.

    표시 정보

  • 시설 아이콘
  • 설치 수량
  • 시설 이름
  • 시설 레벨
  • 시설 설명
  • 시간당 생산량
  • 풍년 효과
  • 이벤트 효과

    • 예시:

    곡물 농장
LV.3
곡물을 생산하는 기본 농장입니다.
생산량: 210 /h
풍년 효과: +20%
이벤트 효과: 봄 축제

    설치 수량

    각 시설 왼쪽의 숫자는 해당 시설이 현재 설치된 수량을 나타낸다.

    같은 시설이 여러 개 설치되어 있으면 설치 수량이 합산되어 표시된다.

    시설 레벨

    시설 이름 아래에는 현재 레벨이 표시된다.

    시설 레벨이 높아지면 생산량이나 시설 효과가 증가할 수 있다.

    시설 설치
→ 시설 레벨 상승
→ 생산량 또는 적용 효과 증가

    생산량

    생산 시설에는 시간당 생산량이 표시된다.

    생산량: 수량 /h

    표시된 수치는 해당 시설의 기본 생산량과 적용 중인 효과가 반영된 현재 생산량이다.

    풍년 효과

    풍년 효과는 시설의 생산 성능을 일시적으로 증가시키는 효과다.

    표시 방식

  • 풍년 효과 아이콘
  • 증가 비율
  • 현재 적용 여부

    • 예시:

    풍년 효과 +20%

    화면 왼쪽 요약에서는 현재 풍년 효과가 적용된 시설 수를 확인할 수 있다.

    이벤트 효과

    시설에 현재 적용 중인 이벤트 효과를 표시한다.

    예시

  • 봄 축제
  • 생산 이벤트
  • 지역 이벤트
  • 사회정책 효과
  • 미생물 이벤트

    • 화면 왼쪽 요약에서는 이벤트 효과가 적용된 시설 수를 확인할 수 있다.

    빌딩 메뉴 진입

→ 풍년 효과와 이벤트 효과 확인

    빌딩

    농장, 공장, 상점, 개미굴, 연구소에 설치된 시설을 확인합니다. 시설별 설치 수량, 레벨, 생산량, 풍년 효과와 이벤트 효과를 한 화면에서 관리할 수 있습니다.
